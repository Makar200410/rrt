Степень окисления и валентность

Важно понимать: Степень окисления и валентность — это разные понятия, хотя их численные значения часто совпадают.

1. Определения и различия

Степень окисления (СО)

Это условный заряд атома в соединении, вычисленный из предположения, что все связи являются ионными (электроны полностью перешли от менее электроотрицательного атома к более электроотрицательному).

Может быть: положительной (+), отрицательной (-), нулевой (0) и даже дробной.

Валентность

Это способность атома образовывать определенное число химических связей. Численно равна количеству связей, которые образует атом.

Обозначается римскими цифрами (I, II, III...). Не имеет знака. Не может быть нулем (в соединениях) или отрицательной.

Пример различия: В молекуле азота N2:

  • Степень окисления N = 0 (простое вещество).
  • Валентность N = III (тройная связь N≡N).

В пероксиде водорода H2O2 (H–O–O–H):

  • Степень окисления O = -1.
  • Валентность O = II (две связи: одна с H, одна с O).

2. Правила определения степени окисления

  1. В простых веществах СО всегда равна 0 (O2, Fe, P4).
  2. Сумма СО всех атомов в нейтральной молекуле равна 0, в ионе — заряду иона.
  3. Постоянная СО:
    • Металлы IA группы (Li, Na, K...): +1
    • Металлы IIA группы (Mg, Ca, Ba...): +2
    • Алюминий (Al): +3
    • Фтор (F): всегда -1
  4. Условно-постоянная СО:
    • Водород (H): +1 (с неметаллами), -1 (с металлами - гидриды).
    • Кислород (O): обычно -2 (оксиды), редко -1 (пероксиды), +2 (OF2).

Примеры нахождения степеней окисления

Для определения степени окисления элемента в сложном веществе можно составить простое уравнение. Пусть x — это искомая степень окисления. Нажмите на пример, чтобы посмотреть решение шаг за шагом.

Пример 1: Перманганат калия (KMnO4)

+
  • У калия (K) как у щелочного металла СО постоянная: +1.
  • У кислорода (O) обычно СО равна: -2.
  • Составим уравнение по правилу электронейтральности (сумма всех СО равна 0):
K+1Mn+xO4-2
(+1) + x + 4 · (-2) = 0
1 + x - 8 = 0
x - 7 = 0
x = +7

Ответ: Степени окисления: K+1Mn+7O4-2.

Пример 2: Дихромат калия (K2Cr2O7)

+
  • У калия (K): +1.
  • У кислорода (O): -2.
  • Пусть у хрома СО равна x. Атомов хрома два, поэтому пишем 2x.
K2+1Cr2+xO7-2
2 · (+1) + 2x + 7 · (-2) = 0
2 + 2x - 14 = 0
2x - 12 = 0
2x = 12
x = +6

Ответ: K2+1Cr2+6O7-2.

Пример 3: Ион аммония (NH4+)

+
  • Это заряженный ион (катион), поэтому сумма СО равна заряду иона: +1.
  • У водорода (H) с неметаллами: +1.
  • Пусть у азота СО равна x.
(NxH4+1)+1
x + 4 · (+1) = +1
x + 4 = 1
x = -3

Ответ: (N-3H4+1)+1.

3. Максимальная степень окисления переходных металлов

У переходных металлов (d-элементов) валентные электроны находятся на внешнем s-подуровне и предвнешнем d-подуровне.

Группа I II III IV V VI VII VIII
Элемент Cu Zn Sc Ti V Cr Mn Fe, Co, Ni
Высшая СО +2 (+3*) +2 +3 +4 +5 +6 +7 Fe (+6, +3, +2)
Co (+3, +2)
Ni (+2)

* Редкие или неустойчивые степени окисления.

7. Валентность и исключения

В большинстве случаев численное значение валентности совпадает с модулем степени окисления. Однако есть важные исключения, которые необходимо знать для ЕГЭ.

Вещество / Частица Элемент Степень окисления Валентность Примечание
N2
(Азот) 		N 0 III Тройная связь N≡N
CO
(Угарный газ) 		C +2 III Тройная связь C≡O
CO
(Угарный газ) 		O -2 III Кислород — донор пары электронов
HNO3
(Азотная кислота) 		N +5 IV Максимальная валентность азота — IV
N2O5
(Оксид азота V) 		N +5 IV Аналогично азотной кислоте
NH4+
(Ион аммония) 		N -3 IV 4 ковалентные связи (одна донорно-акцепторная)
H2O2
(Пероксид водорода) 		O -1 II Две связи: H—O и O—O
H3O+
(Ион гидроксония) 		O -2 III Три связи с водородом

(напр. C2H6) 		C -3 (в C2H6) IV Валентность углерода в органике почти всегда IV
