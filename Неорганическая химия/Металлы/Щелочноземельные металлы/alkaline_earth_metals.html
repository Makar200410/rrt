Щелочноземельные металлы

Химия элементов IIА группы

Важно для ЕГЭ: Щелочноземельные металлы расположены во II группе главной подгруппы (IIА). К ним традиционно относят кальций (Ca), стронций (Sr), барий (Ba) и радий (Ra). Бериллий (Be) и магний (Mg) также входят в эту группу, но имеют ряд отличий. Все элементы группы проявляют постоянную степень окисления +2.

1. Общая характеристика

Элементы группы: Бериллий (Be), Магний (Mg), Кальций (Ca), Стронций (Sr), Барий (Ba), Радий (Ra).

Строение атома

Внешний электронный уровень имеет конфигурацию ns2. Два s-электрона легко отдаются в химических реакциях, поэтому металлы проявляют сильные восстановительные свойства.

Закономерности в группе (Be → Ra)

Физические свойства

Это металлы серого цвета, тверже щелочных металлов. Обладают металлической кристаллической решеткой, высокой тепло- и электропроводностью.

Хранение: Бериллий, Магний и Кальций устойчивы на сухом воздухе (покрываются оксидной пленкой). Стронций и Барий более активны, их хранят под слоем керосина.

2. Нахождение в природе

В природе встречаются исключительно в виде соединений (солей).

Название минерала Формула Химическое название
Доломит CaCO3 · MgCO3 Карбонат кальция-магния
Магнезит MgCO3 Карбонат магния
Кальцит (Мел, Мрамор, Известняк) CaCO3 Карбонат кальция
Гипс CaSO4 · 2H2O Дигидрат сульфата кальция
Барит BaSO4 Сульфат бария
Витерит BaCO3 Карбонат бария

3. Способы получения

1. Электролиз расплавов (для Mg, Ca):

Магний получают из карналлита или хлорида магния (часто с добавкой NaCl для снижения t° плавления):

MgCl2 ⚡→ Mg + Cl2↑

Кальций также получают электролизом расплава хлорида:

CaCl2 ⚡→ Ca + Cl2↑

2. Металлотермия:

Восстановление оксидов более активными металлами или кремнием. Например, получение магния из доломита:

2(CaO · MgO) + Si t°→ 2Mg + Ca2SiO4

Барий получают восстановлением оксида алюминием в вакууме:

4BaO + 2Al t°→ 3Ba + Ba(AlO2)2

4. Химические свойства металлов

Щелочноземельные металлы — сильные восстановители.

Реагент Описание процесса и уравнения реакций
С галогенами Активно реагируют при нагревании, образуя галогениды.
Be + Cl2 → BeCl2
С серой и фосфором При нагревании образуют сульфиды и фосфиды.
Ca + S t°→ CaS
3Ca + 2P t°→ Ca3P2
С водородом При нагревании образуют гидриды (кроме бериллия). Магний реагирует при повышенном давлении.
Mg + H2 → MgH2
С азотом При нагревании образуют нитриды.
3Mg + N2 t°→ Mg3N2
С углеродом Образуют карбиды (ацетилениды для Ca, метаниды для Be).
Ca + 2C t°→ CaC2
2Be + C t°→ Be2C
С кислородом (Горение) Горят на воздухе с образованием оксидов (и частично нитридов).
2Be + O2 900°C→ 2BeO
2Mg + O2 650°C→ 2MgO
С водой • Be не реагирует.
• Mg реагирует только при кипячении:
Mg + 2H2O t°→ Mg(OH)2 + H2↑
• Ca, Sr, Ba реагируют при комнатной температуре (активно):
Ca + 2H2O → Ca(OH)2 + H2↑
С кислотами Вытесняют водород из кислот-неокислителей (HCl, разб. H2SO4).
Mg + 2HCl → MgCl2 + H2↑

С кислотами-окислителями:
С конц. H2SO4 (возможно образование S):
3Ca + 4H2SO4(конц.) → 3CaSO4 + S + 4H2O
С HNO3 (продукты восстановления азота варьируются):
4Ca + 10HNO3(конц.) → 4Ca(NO3)2 + N2O↑ + 5H2O
4Ba + 10HNO3(оч.разб.) → 4Ba(NO3)2 + NH4NO3 + 3H2O
Восстановление оксидов Могут восстанавливать неметаллы (Si, C, B) из их оксидов.
2Ca + SiO2 → 2CaO + Si
Магний горит в углекислом газе:
2Mg + CO2 → 2MgO + C
Вытеснение металлов В расплавах вытесняют менее активные металлы.
Ca + CuCl2(расплав) → CaCl2 + Cu

5.1. Оксиды (MO)

Оксиды Ca, Sr, Ba, Mg — основные. Оксид бериллия BeO — амфотерный.

Получение:

  • Окисление металлов: 2Ca + O2 → 2CaO
  • Разложение карбонатов: CaCO3 t°→ CaO + CO2, MgCO3 → MgO + CO_2
  • Разложение нитратов: 2Ca(NO3)2 t°→ 2CaO + 4NO2 + O2
  • Разложение гидроксидов: Mg(OH)2 t°→ MgO + H2O

Химические свойства:

  1. С кислотными оксидами: MgO + CO2 → MgCO3
  2. С кислотами: CaO + 2HCl → CaCl2 + H2O
  3. С водой:
    • CaO, SrO, BaO реагируют активно (гашение извести): CaO + H2O → Ca(OH)2
    • MgO реагирует только при нагревании: MgO + H2O t°→ Mg(OH)2
    • BeO с водой не реагирует.
  4. Свойства амфотерного BeO:
    • С основными оксидами: BeO + Na2O → Na2BeO2 (бериллат)
    • С щелочами в расплаве: BeO + 2NaOH t°→ Na2BeO2 + H2O
    • С щелочами в растворе: BeO + 2NaOH + H2O → Na2[Be(OH)4]

5.2. Гидроксиды (M(OH)2)

Гидроксиды Ca, Sr, Ba — щелочи (сила основания растет от Ca к Ba). Mg(OH)2 — нерастворимое основание. Be(OH)2 — амфотерный гидроксид.

Получение:

  • Оксид + Вода (для активных): CaO + H2O → Ca(OH)2
  • Металл + Вода: 2Ca + 2H2O → 2Ca(OH)2 + H2↑
  • Обменная реакция (для малорастворимых): Ca(NO3)2 + 2KOH → Ca(OH)2↓ + 2KNO3

Химические свойства:

  1. Взаимодействие с кислотами:
    Гидроксиды кальция, стронция и бария реагируют со всеми кислотами (сильными и слабыми).
    Гидроксид магния взаимодействует только с сильными кислотами.
    Ca(OH)2 + 2HCl → CaCl2 + 2H2O
  2. Взаимодействие с кислотными оксидами:
    Образуются средние или кислые соли (в зависимости от соотношения реагентов).
    Ba(OH)2(изб.) + CO2 → BaCO3↓ + H2O (средняя соль)
    Ba(OH)2 + 2CO2(изб.) → Ba(HCO3)2 (кислая соль)
  3. Взаимодействие с амфотерными оксидами и гидроксидами (для Ca, Sr, Ba):
    • В расплаве (образуются средние соли):
      Ba(OH)2 + Al2O3 t°→ Ba(AlO2)2 + H2O
    • В растворе (образуются комплексные соли):
      Ba(OH)2 + Al2O3 + 3H2O → Ba[Al(OH)4]2
  4. Взаимодействие с кислыми солями:
    Образуются средние или менее кислые соли.
    Ca(OH)2 + Ca(HCO3)2 → 2CaCO3↓ + 2H2O
  5. Взаимодействие с неметаллами:
    Гидроксиды Ca, Sr, Ba взаимодействуют с простыми веществами-неметаллами (кроме инертных газов, N2, O2, H2, C). См. свойства щелочных металлов.
  6. Взаимодействие с амфотерными металлами (кроме Fe, Cr):
    • В расплаве:
      Ca(OH)2 + Zn t°→ CaZnO2 + H2↑
    • В растворе:
      Ca(OH)2 + 2Al + 6H2O → Ca[Al(OH)4]2 + 3H2↑
  7. Обменные реакции с растворимыми солями:
    Реагируют с солями тяжелых металлов, карбонатами, сульфитами, силикатами, сульфатами (для Sr, Ba), солями аммония.
    Ba(OH)2 + FeCl2 → BaCl2 + Fe(OH)2↓
    2NH4Br + Ca(OH)2 → CaBr2 + 2NH3↑ + 2H2O
  8. Термическое разложение:
    Ca(OH)2 (при ~580°C), Mg(OH)2, Be(OH)2 разлагаются при нагревании.
    Mg(OH)2 t°→ MgO + H2O
  9. Сила оснований:
    Гидроксиды Ca, Sr, Ba — сильные основания (щелочи), полностью диссоциируют:
    Ba(OH)2 ⇄ Ba2+ + 2OH-
    Mg(OH)2 — нерастворимое основание. Be(OH)2 — амфотерный гидроксид.
  10. Свойства амфотерного Be(OH)2:
    Взаимодействует с щелочами.
    В расплаве: Be(OH)2 + 2NaOH t°→ Na2BeO2 + 2H2O
    В растворе: Be(OH)2 + 2NaOH → Na2[Be(OH)4]

5.3. Соли и жесткость воды

Разложение нитратов

Нитраты Ca, Sr, Ba разлагаются до нитритов (по одной из схем), Mg — до оксида.

Ca(NO3)2 t°→ Ca(NO2)2 + O2

Примечание: В некоторых условиях или источниках указывается разложение до оксида (как при получении оксидов).

Карбонаты

  • Термическое разложение: CaCO3 1200°C→ CaO + CO2
  • Взаимодействие с сильными кислотами: CaCO3 + 2HCl → CaCl2 + CO2↑ + H2O
  • Превращение в гидрокарбонаты (растворение в воде с CO2):
    CaCO3 + H2O + CO2 → Ca(HCO3)2
  • Сплавление с менее летучими оксидами (SiO2, Al2O3):
    CaCO3 + Al2O3 t°→ Ca(AlO2)2 + CO2↑

Жесткость воды

Обусловлена наличием солей кальция и магния.

Временная (карбонатная) жесткость: Гидрокарбонаты Ca(HCO3)2 и Mg(HCO3)2.

Постоянная (некарбонатная) жесткость: Хлориды и сульфаты (CaCl2, MgSO4 и др.).

Способы устранения:

  1. Кипячение (только для временной):
    Ca(HCO3)2 t°→ CaCO3↓ + CO2↑ + H2O
  2. Добавление извести (для временной):
    Ca(HCO3)2 + Ca(OH)2 → 2CaCO3↓ + 2H2O
  3. Добавление соды (для постоянной):
    CaCl2 + Na2CO3 → CaCO3↓ + 2NaCl
  4. Добавление фосфатов (для постоянной):
    3CaCl2 + 2Na3PO4 → Ca3(PO4)2↓ + 6NaCl

6. Качественные реакции

Окрашивание пламени

  • Кальций (Ca2+): Кирпично-красный цвет
  • Стронций (Sr2+): Карминово-красный (алый)
  • Барий (Ba2+): Яблочно-зеленый

Осаждение ионов

  • На ионы Mg2+: Взаимодействие с щелочами (белый осадок):
    Mg2+ + 2OH- → Mg(OH)2↓
  • На ионы Ca2+, Sr2+, Ba2+: Взаимодействие с карбонатами (белый осадок):
    Ca2+ + CO32- → CaCO3↓
    Ba2+ + CO32- → BaCO3↓
  • На ионы Sr2+, Ba2+: Взаимодействие с сульфатами (белый осадок):
    Ba2+ + SO42- → BaSO4↓
  • Фосфаты также дают белые осадки:
    3Ca2+ + 2PO43- → Ca3(PO_4)2↓

7. Видео эксперименты

  • Магний (Magnesium)
  • Кальций (Calcium)
  • Стронций (Strontium)
  • Барий (Barium)
