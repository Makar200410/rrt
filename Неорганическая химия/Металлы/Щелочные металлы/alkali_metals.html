Щелочные металлы

Химия элементов IА группы: от Лития до Франция

Важно для ЕГЭ: Щелочные металлы — самые активные металлы. В соединениях они всегда проявляют постоянную степень окисления +1. Их гидроксиды являются растворимыми в воде сильными основаниями (щелочами).

1. Общая характеристика элементов IA группы

К щелочным металлам относятся элементы главной подгруппы I группы Периодической системы:

  • Литий (Li)
  • Натрий (Na)
  • Калий (K)
  • Рубидий (Rb)
  • Цезий (Cs)
  • Франций (Fr) — радиоактивный элемент

Строение атома

Все эти элементы имеют сходное строение внешнего электронного уровня:

ns1

На внешнем уровне находится всего один валентный s-электрон, который они легко отдают в химических реакциях, проявляя свойства сильных восстановителей.

Закономерности в группе (Li → Fr)

Свойство Изменение сверху вниз
Атомный радиус Увеличивается (растет число электронных слоев)
Металлические свойства Усиливаются
Восстановительная способность Усиливается
Электроотрицательность Уменьшается
Температура плавления Уменьшается

Физические свойства

Щелочные металлы — это мягкие вещества серебристо-белого цвета (на свежем срезе). Их можно резать ножом. Обладают характерным металлическим блеском, высокой тепло- и электропроводностью. Имеют низкие температуры плавления и низкую плотность (литий, натрий и калий легче воды).

Из-за высокой химической активности их хранят под слоем керосина или в запаянных ампулах (чтобы исключить контакт с кислородом и влагой воздуха).

2. Нахождение в природе

Ввиду своей исключительной активности, в свободном виде щелочные металлы в природе не встречаются. Они существуют только в виде соединений (солей).

Название минерала Формула Химическое название
Каменная соль (Галит) NaCl Хлорид натрия
Сильвин KCl Хлорид калия
Сильвинит NaCl · KCl Смешанный хлорид натрия-калия
Глауберова соль (Мирабилит) Na2SO4 · 10H2O Декагидрат сульфата натрия
Селитра чилийская NaNO3 Нитрат натрия

3. Способы получения

Основной промышленный метод получения щелочных металлов — электролиз расплавов их солей (чаще всего хлоридов).

1. Получение лития и натрия (Электролиз):

В расплаве ионы металла восстанавливаются на катоде:

2NaCl (расплав) → 2Na + Cl2↑

На катоде (-): Na+ + e- → Na0
На аноде (+): 2Cl- - 2e- → Cl20

Аналогично получают литий из расплава LiCl.

2. Термохимическое восстановление (для K, Rb, Cs):

Более тяжелые металлы (калий, рубидий, цезий) трудно получить электролизом из-за их высокой летучести и растворимости в расплавленных солях. Их вытесняют из расплавов солей более активными (или менее летучими) металлами, либо восстанавливают:

KCl + Na ⇄ NaCl + K↑

2CsCl + Ca → CaCl2 + 2Cs↑

4. Химические свойства простых веществ (Металлов)

Щелочные металлы — сильнейшие восстановители. Они легко отдают единственный внешний электрон, превращаясь в положительно заряженные ионы M+.

Реагент Описание процесса и уравнения реакций
С галогенами Реагируют очень бурно (со взрывом или воспламенением), образуя галогениды (ионные соли).
2Na + Cl2 → 2NaCl
2K + I2 → 2KI
С серой При нагревании (с ртутью, а также калий с серой могут реагировать даже при растирании) образуют сульфиды.
2Na + S t°→ Na2S
С фосфором Образуют фосфиды при нагревании.
3K + P t°→ K3P
С водородом Образуют твердые ионные гидриды (активные восстановители).
2Na + H2 t°→ 2NaH
С азотом Литий реагирует с азотом при комнатной температуре (уникальное свойство!), остальные — при нагревании. Образуются нитриды.
6Li + N2 → 2Li3N
С кислородом
Важное различие!
При горении на воздухе:
• Литий образует нормальный оксид: 4Li + O2 → 2Li2O
• Натрий образует пероксид: 2Na + O2 → Na2O2
• Калий, Рубидий, Цезий образуют надпероксиды: K + O2 → KO2
Цезий на воздухе самовозгорается мгновенно.
С водой Реагируют бурно (натрий "бегает" по воде, калий загорается, цезий взрывается). Образуется щелочь и выделяется водород.
2Na + 2H2O → 2NaOH + H2↑
С кислотами Реакция протекает со взрывом. Проводить опасно.
С HCl: 2Na + 2HCl → 2NaCl + H2↑
С конц. H2SO4 (возможно образование H2S и S):
8Na + 5H2SO4(конц.) → 4Na2SO4 + H2S + 4H2O

Взаимодействие с органическими веществами

Щелочные металлы способны замещать водород в соединениях, обладающих даже очень слабыми кислотными свойствами:

  • Со спиртами (образование алкоголятов):
    2CH3OH + 2Na → 2CH3ONa + H2↑
  • С фенолом (образование фенолятов):
    2C6H5OH + 2Na → 2C6H5ONa + H2↑
  • С алкинами (с концевой тройной связью):
    HC≡CH + 2Na → NaC≡CNa + H2↑
  • С галогеналканами (реакция Вюрца):
    2CH3Cl + 2Na → C2H6 + 2NaCl

5.1. Оксиды щелочных металлов (M2O)

Оксиды (например, Na2O, K2O) — это твердые вещества, проявляющие ярко выраженные основные свойства.

Получение: Прямым окислением можно получить только Li2O. Оксиды других металлов получают косвенно:

  • Нагревание пероксида с металлом: Na2O2 + 2Na → 2Na2O
  • Нагревание нитрата с металлом: 2NaNO3 + 10Na → 6Na2O + N2↑

Химические свойства:

  1. С водой (очень активно): Na2O + H2O → 2NaOH
  2. С кислотными оксидами: Na2O + CO2 → Na2CO3
  3. С кислотами: K2O + 2HCl → 2KCl + H2O
  4. Окисление кислородом (кроме Li2O): 2Na2O + O2 → 2Na2O2

5.2. Пероксиды щелочных металлов (M2O2)

Содержат кислород в степени окисления -1 (группа -O-O-). Пероксид натрия Na2O2 — желтоватый порошок.

Свойства:

  • Реакция с водой (гидролиз):
    • На холоде: Na2O2 + 2H2O → 2NaOH + H2O2
    • При нагревании (диспропорционирование): 2Na2O2 + 2H2O → 4NaOH + O2↑
  • С углекислым газом (регенерация воздуха на подводных лодках):
    2Na2O2 + 2CO2 → 2Na2CO3 + O2↑
  • Окислительно-восстановительная двойственность:
    • Как окислитель: Na2O2 + SO2 → Na2SO4
    • Как восстановитель (с сильным окислителем): 5Na2O2 + 2KMnO4 + 8H2SO4 → 5O2↑ + 2MnSO4 + 5Na2SO4 + K2SO4 + 8H2O

5.3. Гидроксиды (Щелочи)

Гидроксиды щелочных металлов (LiOH, NaOH, KOH, RbOH, CsOH) — твердые белые вещества, гигроскопичные, прекрасно растворимые в воде. Являются сильными основаниями (щелочами).

Тривиальные названия: NaOH — едкий натр (каустическая сода), KOH — едкое кали.

Химические свойства:

  1. Изменяют окраску индикаторов (лакмус — синий, фенолфталеин — малиновый, метилоранж — желтый).
  2. С кислотами (реакция нейтрализации): NaOH + HCl → NaCl + H2O (в т.ч. с органическими кислотами).
  3. С кислотными оксидами: 2KOH + CO2 → K2CO3 + H2O
  4. С амфотерными оксидами и гидроксидами:
    • В расплаве: 2NaOH + Al2O3 t°→ 2NaAlO2 + H2O (метаалюминат)
    • В растворе: 2NaOH + Al2O3 + 3H2O → 2Na[Al(OH)4] (тетрагидроксоалюминат)
  5. С солями (если образуется осадок или газ):
    CuSO4 + 2NaOH → Cu(OH)2↓ + Na2SO4
    NH4Cl + NaOH t°→ NaCl + NH3↑ + H2O
  6. С неметаллами (диспропорционирование):
    • Хлор на холоде: Cl2 + 2KOH → KCl + KClO + H2O
    • Хлор при нагревании: 3Cl2 + 6KOH → 5KCl + KClO3 + 3H2O
    • Кремний (окисление Si): Si + 2NaOH + H2O → Na2SiO3 + 2H2↑
  7. Разложение при нагревании:
    Гидроксиды щелочных металлов плавятся без разложения, ИСКЛЮЧЕНИЕ — LiOH:
    2LiOH t°→ Li2O + H2O

5.4. Соли щелочных металлов

Почти все соли щелочных металлов растворимы в воде. Особый интерес представляет термическое разложение нитратов.

Разложение нитратов при нагревании:

1. Нитрат лития (как и нитраты магния, меди) разлагается до оксида:

4LiNO3 → 2Li2O + 4NO2 + O2

2. Нитраты остальных щелочных металлов (Na, K, Rb, Cs) разлагаются до нитритов:

2NaNO3 → 2NaNO2 + O2

2KNO3 → 2KNO2 + O2

6. Качественные реакции (Окрашивание пламени)

Ионы щелочных металлов окрашивают пламя горелки в характерные цвета, что используется для их качественного обнаружения.

Ион металла Цвет пламени
Li+ Карминно-красный
Na+ Желтый
K+ Фиолетовый
Rb+ Буро-красный
Cs+ Фиолетово-красный
