Хром

Элемент побочной подгруппы VI группы: мастер изменения цветов

Важно для ЕГЭ: Хром проявляет переменные степени окисления: +2, +3, +6. Соединения Cr(III) — амфотерны, Cr(VI) — сильные окислители и проявляют кислотные свойства.

1. Строение и положение в ПС

Хром (Cr) находится в 4-м периоде, VI группе (побочной подгруппе). Порядковый номер 24.

Особенность электронной конфигурации

У атома хрома наблюдается "проскок" электрона с 4s-подуровня на 3d-подуровень, что обеспечивает более устойчивое состояние (наполовину заполненный d-подуровень).

...3d54s1

2. Физические свойства

Хром — твердый металл голубовато-белого цвета. Один из самых твердых металлов (режет стекло). Тугоплавкий (t°пл ≈ 1900°C), устойчив к коррозии на воздухе.

3. Способы получения

В природе встречается в виде минерала хромистого железняка (FeO·Cr2O3).

1. Алюмотермия (основной способ получения чистого хрома):

Cr2O3 + 2Al t°→ 2Cr + Al2O3

2. Восстановление коксом (получение феррохрома):

FeO·Cr2O3 + 4C t°→ Fe + 2Cr + 4CO↑

4. Химические свойства простого вещества

При обычных условиях хром химически инертен из-за прочной оксидной пленки (пассивация). Реагирует только при нагревании.

Взаимодействие с неметаллами

С галогенами 2Cr + 3Cl2 t°→ 2CrCl3
С кислородом Горит с образованием оксида хрома (III):
4Cr + 3O2 t°→ 2Cr2O3
С серой 2Cr + 3S t°→ Cr2S3
С азотом 2Cr + N2 1000°C→ 2CrN

Взаимодействие со сложными веществами

1. С водой:
В раскаленном состоянии реагирует с водяным паром:

2Cr + 3H2O(пар) t°→ Cr2O3 + 3H2

2. С кислотами-неокислителями (HCl, разб. H2SO4):
Хром вытесняет водород, образуя соли двухвалентного хрома (синие растворы):

Cr + 2HCl → CrCl2 + H2↑

В присутствии кислорода воздуха Cr(II) быстро окисляется до Cr(III):
4CrCl2 + 4HCl + O2 → 4CrCl3 + 2H2O

3. С кислотами-окислителями:
Концентрированные H2SO4 и HNO3 пассивируют хром на холоде. Реакция идет только при нагревании, образуются соли Cr(III):

2Cr + 6H2SO4(конц.) t°→ Cr2(SO4)3 + 3SO2↑ + 6H2O

Cr + 6HNO3(конц.) t°→ Cr(NO3)3 + 3NO2↑ + 3H2O

4. С солями (вытеснение металлов):

2Cr + 3CuCl2 → 2CrCl3 + 3Cu

5. С окислителями в щелочной среде:
Хром переходит в степень окисления +6 (хроматы):

Cr + KClO3 + 2KOH t°→ K2CrO4 + KCl + H2O

5. Оксиды хрома

5.1. Оксид хрома (II) — CrO (Основный)

Черный порошок. Проявляет основные свойства и является сильным восстановителем.

  • Реакция с кислотами: CrO + 2HCl → CrCl2 + H2O
  • Окисление на воздухе: 4CrO + O2 t°→ 2Cr2O3
  • Диспропорционирование: 3CrO t°→ Cr + Cr2O3

5.2. Оксид хрома (III) — Cr2O3 (Амфотерный)

Тугоплавкий порошок зеленого цвета. Химически инертен.

Получение:

  • Разложение гидроксида: 2Cr(OH)3 t°→ Cr2O3 + 3H2O
  • Вулкан Бёттгера (разложение дихромата аммония):
    (NH4)2Cr2O7 t°→ Cr2O3 + N2↑ + 4H2O

Химические свойства:

1. С кислотами (при нагревании):
Cr2O3 + 6HCl t°→ 2CrCl3 + 3H2O
Cr2O3 + 3H2SO4 t°→ Cr2(SO4)3 + 3H2O

2. С щелочами и карбонатами (сплавление):
Cr2O3 + 2NaOH t°→ 2NaCrO2 + H2O
Cr2O3 + Na2CO3 t°→ 2NaCrO2 + CO2↑

3. Окисление в щелочной среде и с карбонатами (переход в хроматы):
В присутствии сильных окислителей Cr2O3 проявляет восстановительные свойства.
С нитратами и щелочью:
Cr2O3 + 3KNO3 + 4KOH t°→ 2K2CrO4 + 3KNO2 + 2H2O
С нитратами и карбонатами:
Cr2O3 + 3NaNO3 + 2Na2CO3 t°→ 2Na2CrO4 + 3NaNO2 + 2CO2↑
С хлоратами и карбонатами:
Cr2O3 + KClO3 + 2Na2CO3 t°→ 2Na2CrO4 + KCl + 2CO2↑
С бромом в щелочи:
Cr2O3 + 3Br2 + 10NaOH → 2Na2CrO4 + 6NaBr + 5H2O

4. Алюмотермия (восстановление металла):
Cr2O3 + 2Al t°→ Al2O3 + 2Cr

5.3. Оксид хрома (VI) — CrO3 (Кислотный)

Темно-красные кристаллы, гигроскопичен, ядовит. Сильный окислитель.

Получение:

Na2Cr2O7 + H2SO4(конц.) → 2CrO3 + 2NaHSO4 + H2O

Свойства:

  • Растворяется в воде с образованием кислот:
    CrO3 + H2O → H2CrO4 (хромовая)
    2CrO3 + H2O → H2Cr2O7 (дихромовая)
  • Окислительные свойства (окисляет спирт, серу, фосфор):
    2CrO3 + C2H5OH → 2CrO2 + CH3CHO + H2O (качественная реакция на этанол -- трубка алкотестера)

6. Гидроксиды хрома

6.1. Гидроксид хрома (II) — Cr(OH)2 (Основный)

Желтый осадок. Проявляет основные свойства. Неустойчив, сильный восстановитель.

Получение:

Действие щелочей на солей Cr(II) (в инертной атмосфере):
CrCl2 + 2NaOH → Cr(OH)2↓ + 2NaCl

Химические свойства:

  • Как основание реагирует с кислотами:
    Cr(OH)2 + 2HCl → CrCl2 + 2H2O
  • Быстро окисляется на воздухе (превращается в Cr(OH)3):
    4Cr(OH)2 + O2 + 2H2O → 4Cr(OH)3
    (Наблюдается изменение цвета с желтого на серо-зеленый)

6.2. Гидроксид хрома (III) — Cr(OH)3 (Амфотерный)

Серо-зеленый студенистый осадок. Типичный амфотерный гидроксид.

Получение:

  • Действие аммиака (лучший способ):
    CrCl3 + 3NH3·H2O → Cr(OH)3↓ + 3NH4Cl
    (Осадок не растворяется в избытке аммиака)
  • Взаимный гидролиз (соль Cr3+ + карбонат/сульфид):
    2CrCl3 + 3Na2CO3 + 3H2O → 2Cr(OH)3↓ + 3CO2↑ + 6NaCl
    2CrCl3 + 3Na2S + 6H2O → 2Cr(OH)3↓ + 3H2S↑ + 6NaCl
  • Разрушение комплексов (действие CO2):
    Na3[Cr(OH)6] + 3CO2 → Cr(OH)3↓ + 3NaHCO3

Химические свойства:

  • С кислотами (как основание):
    Cr(OH)3 + 3HNO3 → Cr(NO3)3 + 3H2O
  • С щелочами (как кислота):
    В растворе (образуются гидроксокомплексы):
    Cr(OH)3 + 3NaOH → Na3[Cr(OH)6] (гексагидроксохромат)
    В расплаве (образуются хромиты):
    Cr(OH)3 + KOH t°→ KCrO2 + 2H2O
  • Термическое разложение:
    2Cr(OH)3 t°→ Cr2O3 + 3H2O
  • Окисление (в щелочной среде):
    2Cr(OH)3 + 3Br2 + 10KOH → 2K2CrO4 + 6KBr + 8H2O
    2Cr(OH)3 + 3H2O2 + 4NaOH → 2Na2CrO4 + 8H2O

7. Соли хрома

Переходы между хроматами и дихроматами

В зависимости от среды, Cr(VI) существует в разных формах:

Хроматы (CrO42-) — устойчивы в щелочной среде. Желтый цвет.
Дихроматы (Cr2O72-) — устойчивы в кислой среде. Оранжевый цвет.

Реакция перехода ("Оранжевое → Желтое"):

K2Cr2O7 + 2KOH → 2K2CrO4 + H2O

Реакция перехода ("Желтое → Оранжевое"):

2K2CrO4 + H2SO4 → K2Cr2O7 + K2SO4 + H2O

Окислительные свойства соединений Cr(VI)

Хроматы и дихроматы — сильные окислители. Продукты восстановления зависят от среды:

Среда Продукт восстановления Цвет раствора
Кислая (H+) Соли Cr3+ Сине-фиолетовый / Зеленый
Нейтральная (H2O) Cr(OH)3 Серо-зеленый осадок
Щелочная (OH-) [Cr(OH)6]3- Изумрудно-зеленый раствор

Примеры реакций:

1. В кислой среде (переход Cr+6 → Cr+3, синий/зеленый):
С сульфитом натрия:
K2Cr2O7 + 3Na2SO3 + 4H2SO4 → Cr2(SO4)3 + 3Na2SO4 + K2SO4 + 4H2O
С галогеноводородами (HCl, HBr):
K2Cr2O7 + 14HCl → 2CrCl3 + 3Cl2↑ + 2KCl + 7H2O
С солями железа (II):
K2Cr2O7 + 6FeSO4 + 7H2SO4 → Cr2(SO4)3 + 3Fe2(SO4)3 + K2SO4 + 7H2O
С сероводородом:
K2Cr2O7 + 3H2S + 4H2SO4 → Cr2(SO4)3 + 3S↓ + K2SO4 + 7H2O

2. В нейтральной среде (переход Cr+6 → Cr(OH)3↓, серо-зеленый осадок):
С сульфидом аммония (или натрия):
K2Cr2O7 + 3(NH4)2S + H2O → 2Cr(OH)3↓ + 3S↓ + 6NH3↑ + 2KOH
С сульфитом натрия:
2K2CrO4 + 3Na2SO3 + 5H2O → 2Cr(OH)3↓ + 3Na2SO4 + 4KOH

3. В щелочной среде (переход Cr+6 → [Cr(OH)6]3-, изумрудный раствор):
С сульфитом натрия:
2K2CrO4 + 3Na2SO3 + 2H2O → 2K3[Cr(OH)6] + 3Na2SO4 + 2KOH
С нитритом калия:
2K2CrO4 + 3KNO2 + 5H2O → 2K3[Cr(OH)6] + 3KNO3 + 2KOH

8. Качественные реакции

  • Cr2+: Голубой раствор → с щелочью дает коричневый осадок Cr(OH)2, буреющий на воздухе.
  • Cr3+: Фиолетовый/зеленый раствор → с щелочью дает серо-зеленый осадок Cr(OH)3, растворимый в избытке щелочи (изумрудный раствор).
  • CrO42-: Желтый раствор, образует нерастворимые осадки с Ba2+ (желтый), Ag+ (красно-бурый).
    K2CrO4 + BaCl2 → BaCrO4↓ + 2KCl
