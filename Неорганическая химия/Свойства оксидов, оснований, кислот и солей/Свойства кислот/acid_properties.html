Кислоты: Классификация, Свойства, Получение

В этом разделе: Подробный разбор классификации, физических и химических свойств, а также способов получения кислот.

1. Классификация кислот

Кислоты — это сложные вещества, молекулы которых состоят из одного или нескольких атомов водорода и кислотного остатка, способные замещаться на атомы металлов с образованием солей.

По наличию кислорода:

  • Бескислородные: HCl, HBr, HI, HF, H₂S.
  • Кислородсодержащие: H₂SO₄, HNO₃, H₃PO₄, H₂SiO₃.

По основности (числу атомов водорода):

  • Одноосновные: HCl, HNO₃, HF, CH₃COOH.
  • Двухосновные: H₂SO₄, H₂S, H₂CO₃.
  • Трехосновные: H₃PO₄, H₃AsO₄.

По силе (степени диссоциации):

  • Сильные кислоты: HCl, HBr, HI, HNO₃, H₂SO₄, HClO₄.
  • Слабые кислоты: HF, H₂S, H₂CO₃, H₂SO₃, H₂SiO₃, HNO₂, органические кислоты (CH₃COOH).

По стабильности:

  • Стабильные: H₂SO₄, HCl, H₃PO₄.
  • Нестабильные (разлагаются при н.у. или нагревании): H₂CO₃, H₂SO₃, H₂SiO₃ (при нагревании).

2. Физические свойства

При обычных условиях кислоты могут быть твердыми или жидкими. Газообразные вещества (HCl, H₂S) проявляют кислотные свойства только в водном растворе.

Агрегатное состояние Примеры и описание
Жидкости H₂SO₄ (серная) — тяжелая маслянистая жидкость, бесцветная.
HNO₃ (азотная) — бесцветная, дымящаяся на воздухе жидкость с резким запахом.
HCl, HBr, HI, HF — растворы соответствующих газов в воде (бесцветные жидкости).
Твердые вещества H₃PO₄ (фосфорная) — бесцветные кристаллы, хорошо растворимые в воде.
H₃BO₃ (борная) — белые чешуйки.
H₂SiO₃ (кремниевая) — студенистый осадок, нерастворима в воде.
Цвет Большинство кислот бесцветны.
H₂CrO₄ (хромовая) — желтая.
H₂MnO₄ (марганцоватая) — зеленая.
HMnO₄ (марганцовая) — малиново-фиолетовая.

3. Получение кислот

  • 1. Взаимодействие неметаллов с водородом (для бескислородных кислот):
    H₂ + Cl₂ hν→ 2HCl (газ) → растворение в воде → соляная кислота
    H₂ + S t°→ H₂S
  • 2. Взаимодействие кислотных оксидов с водой (для кислородсодержащих кислот):
    SO₃ + H₂O → H₂SO₄
    P₂O₅ + 3H₂O t°→ 2H₃PO₄
    N₂O₅ + H₂O → 2HNO₃
    Исключение: SiO₂ не реагирует с водой.
  • 3. Вытеснение более сильной или менее летучей кислотой из солей:
    2NaCl(тв.) + H₂SO₄(конц.) t°→ Na₂SO₄ + 2HCl↑
    CaCO₃ + 2HCl → CaCl₂ + CO₂↑ + H₂O (сильная вытесняет слабую)
    Na₂SiO₃ + 2HCl → 2NaCl + H₂SiO₃↓
  • 4. Окисление неметаллов или их соединений:
    3P + 5HNO₃ + 2H₂O → 3H₃PO₄ + 5NO
    Cl₂ + H₂O ⇄ HCl + HClO

4. Химические свойства кислот

Кислоты проявляют общие свойства, обусловленные наличием ионов водорода H⁺.

Свойство Реакция
1. Диссоциация в растворе HCl → H⁺ + Cl⁻ (сильные полностью)
H₂S ⇄ H⁺ + HS⁻ (слабые обратимо)
Многоосновные кислот диссоциируют ступенчато:
H₂SO₃ ⇄ H⁺ + HSO₃⁻ (1 ступень)
HSO₃⁻ ⇄ H⁺ + SO₃²⁻ (2 ступень)

Изменение окраски индикаторов:
Лакмус: красный
Метилоранж: красный
Фенолфталеин: бесцветный
2. Взаимодействие с металлами
(стоящими до H)		 Минеральные кислоты (HCl, H₂SO₄ разб., H₃PO₄) выделяют водород:
Mg + H₂SO₄ → MgSO₄ + H₂↑
Zn + 2HCl → ZnCl₂ + H₂↑
Fe + 2HCl → FeCl₂ + H₂↑ (Fe окисляется до +2)
Исключение: HNO₃ и H₂SO₄(конц.) - см. специфические свойства.
3. Взаимодействие с основными оксидами Основный оксид + Кислота → Соль + Вода
CuO + 2HCl → CuCl₂ + H₂O
CaO + 2HNO₃ → Ca(NO₃)₂ + H₂O
4. Взаимодействие с основаниями
(Реакция нейтрализации)

C образованием средних солей (избыток щелочи):
NaOH + HCl → NaCl + H₂O
3NaOH + H₃PO₄ → Na₃PO₄ + 3H₂O

C образованием кислых солей (избыток многоосновной кислоты):
Зависит от соотношения реагентов:
NaOH + H₃PO₄ (1:1) → NaH₂PO₄ + H₂O (дигидрофосфат)
2NaOH + H₃PO₄ (2:1) → Na₂HPO₄ + 2H₂O (гидрофосфат)

5. Взаимодействие с амфотерными оксидами и гидроксидами Al₂O₃ + 3H₂SO₄ → Al₂(SO₄)₃ + 3H₂O
Zn(OH)₂ + 2HBr → ZnBr₂ + 2H₂O
Амфотерные соединения в реакциях с кислотами проявляют основные свойства.
6. Взаимодействие с солями

Со средними солями (вытеснение слабой/летучей кислоты):
K₂CO₃ + 2HCl → 2KCl + CO₂↑ + H₂O
CaCO₃ + 2HCl → CaCl₂ + CO₂↑ + H₂O

С кислыми солями:
1. Вытеснение слабой кислоты:
KHCO₃ + HCl → KCl + CO₂↑ + H₂O
2. Образование более кислой соли:
K₂HPO₄ + H₃PO₄ → 2KH₂PO₄

С основными солями (переход в среднюю соль):
(CuOH)₂CO₃ + 2H₂SO₄ → 2CuSO₄ + CO₂↑ + 3H₂O
Al(OH)Cl₂ + HCl → AlCl₃ + H₂O

С растворимыми солями (реакция ионного обмена с выпадением осадка):
AgNO₃ + HCl → AgCl↓ + HNO₃
BaCl₂ + H₂SO₄ → BaSO₄↓ + 2HCl
7. Окислительно-восстановительные свойства

Кислоты-окислители (HNO₃, H₂SO₄ конц.):
Cu + 4HNO₃(конц.) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
C + 2H₂SO₄(конц.) → CO₂↑ + 2SO₂↑ + 2H₂O
Окисляют металлы (после H), неметаллы (P, S, C) и сложные вещества.

Кислоты-восстановители (HI, H₂S, HBr, HCl):
Содержат неметалл в низшей или промежуточной с.о., могут окисляться:
2HI + Cl₂ → 2HCl + I₂↓
2H₂S + SO₂ → 3S↓ + 2H₂O (конпропорционирование)
2KMnO₄ + 16HCl → 2MnCl₂ + 5Cl₂↑ + 2KCl + 8H₂O

8. Взаимодействие с аммиаком NH₃ + HCl → NH₄Cl (хлорид аммония)
2NH₃ + H₂SO₄ → (NH₄)₂SO₄
NH₃ + H₃PO₄ → NH₄H₂PO₄ (возможно образование кислых солей)
9. Термическое разложение

Без нагревания (неустойчивые):
H₂CO₃ ⇄ H₂O + CO₂↑
H₂SO₃ ⇄ H₂O + SO₂↑
2HNO₂ ⇄ NO↑ + NO₂↑ + H₂O

При нагревании:
H₂SiO₃ t°→ H₂O + SiO₂
4HNO₃ t°→ 4NO₂ + O₂ + 2H₂O
2HI t°→ H₂ + I₂
10. Взаимодействие с бинарными соединениями

1. Нитриды (гидролиз в кислой среде):
Ca₃N₂ + 8HCl → 3CaCl₂ + 2NH₄Cl

2. Фосфиды:
Zn₃P₂ + 6HCl → 3ZnCl₂ + 2PH₃↑

3. Карбиды (метаниды):
Al₄C₃ + 12HCl → 4AlCl₃ + 3CH₄↑

4. Карбиды (ацетилениды):
CaC₂ + 2HCl → CaCl₂ + C₂H₂↑

5. Силициды:
Mg₂Si + 4HCl → 2MgCl₂ + SiH₄↑
11. Специфические свойства

Взаимодействие SiO₂ с плавиковой кислотой HF:
SiO₂ + 4HF(газ) → SiF₄↑ + 2H₂O (травление стекла)
SiO₂ + 6HF(жидк.) → H₂[SiF₆] + 2H₂O
