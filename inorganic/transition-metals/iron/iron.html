Железо

Элемент побочной подгруппы VIII группы: самый важный металл цивилизации

Важно для ЕГЭ: Железо проявляет две основные степени окисления: +2 и +3. Fe(OH)2 — основание, Fe(OH)3 — амфотерный гидроксид (с преобладанием основных свойств). Железо пассивируется холодными концентрированными H2SO4 и HNO3, но реагирует с ними при нагревании. Соли Fe3+ — окислители!

1. Строение и положение в ПС

Железо (Fe) расположено в 4-м периоде, в побочной подгруппе VIII группы периодической системы. Порядковый номер — 26, атомная масса — 56.

Электронная конфигурация

+26Fe: 1s22s22p63s23p63d64s2

На d-подуровне у атома железа расположено 6 электронов. При отдаче двух 4s-электронов образуется ион Fe2+ с конфигурацией 3d6. Если дополнительно отдать один d-электрон, получается ион Fe3+ (3d5) — конфигурация с наполовину заполненным d-подуровнем, что объясняет относительную устойчивость степени окисления +3. Железо обладает ярко выраженными магнитными свойствами — оно является ферромагнетиком.

Интересный факт: Железо — один из самых распространённых элементов во Вселенной. Ядро нашей планеты на ~85% состоит из железа. Именно железо в составе гемоглобина придаёт крови красный цвет и переносит кислород по организму.

2. Физические свойства

Железо — серебристо-белый металл с высокой ковкостью и пластичностью. Обладает хорошей тепло- и электропроводностью.

  • Температура плавления: 1538°C — один из самых тугоплавких d-металлов.
  • Температура кипения: 2861°C.
  • Магнитные свойства: железо — классический ферромагнетик, притягивается к магниту. Это свойство используется для отделения железа от других веществ.
  • Коррозия: на влажном воздухе покрывается рыхлой ржавчиной (в отличие от алюминия и хрома, у которых оксидная плёнка защищает металл).
  • Применение: основа чёрной металлургии — из железа и его сплавов (стали, чугуна) производят около 95% всех металлических изделий в мире.

3. Нахождение в природе

По распространённости в земной коре железо занимает 4-е место среди всех элементов и 2-е место среди металлов (после алюминия). Содержание в земной коре — около 4-8%.

Основные минералы железа:

  • Красный железняк (гематит) — Fe2O3. Важнейшая руда для выплавки стали.
  • Магнитный железняк (магнетит) — Fe3O4 (или FeO·Fe2O3). Обладает магнитными свойствами.
  • Пирит (серный колчедан) — FeS2. Используется для получения серной кислоты, а не железа, так как при обжиге образуется хрупкое металлическое железо с большим содержанием примесей.

4. Способы получения

Железо в промышленности получают преимущественно из гематита (Fe2O3) и магнетита (Fe3O4).

1. Доменный процесс (основной промышленный способ):

В доменную печь загружают шихту — смесь железной руды, каменноугольного кокса и флюсов (например, известняка CaCO3). Процесс многоэтапный:

Сначала кокс сгорает, образуя угарный газ:

2C + O2 → 2CO

В верхней части печи (200–700°C) начинается постепенное восстановление:

3Fe2O3 + CO → 2Fe3O4 + CO2

Ниже (≈850°C) происходит дальнейшее восстановление:

Fe3O4 + CO → 3FeO + CO2

В самой горячей зоне (до 1200°C) — окончательное восстановление до металла:

FeO + CO → Fe + CO2

Углекислый газ, поднимаясь, реагирует с коксом, регенерируя угарный газ:

CO2 + C → 2CO

Суммарная реакция восстановления:

3CO + Fe2O3 t°→ 2Fe + 3CO2

Важно для ЕГЭ: Флюс (CaCO3) при разложении даёт CaO, который связывает примеси пустой породы, образуя легкоплавкий шлак. Шлак всплывает и защищает расплавленный металл от окисления.

2. Прямое восстановление водородом:

Даёт более чистое железо без примесей серы и фосфора:

Fe2O3 + 3H2 t°→ 2Fe + 3H2O

3. Электролиз растворов солей железа — для получения особо чистого металла.

5. Качественные реакции

На ионы Fe2+

  • Со щелочами — образуется серо-зелёный студенистый осадок гидроксида железа (II), который на воздухе быстро буреет, окисляясь до Fe(OH)3:
    FeCl2 + 2NaOH → Fe(OH)2↓ + 2NaCl
    4Fe(OH)2 + O2 + 2H2O → 4Fe(OH)3
  • Растворы солей Fe2+ имеют светлый жёлто-зелёный цвет.
  • С красной кровяной солью K3[Fe(CN)6] — образуется синий осадок «турнбулева синь».
Fe(OH)2 — серо-зелёный студенистый осадок → буреет на воздухе.

На ионы Fe3+

  • Со щелочами — образуется бурый осадок гидроксида железа (III):
    FeCl3 + 3NaOH → Fe(OH)3↓ + 3NaCl
  • Растворы солей Fe3+ имеют жёлто-оранжевый цвет.
  • С жёлтой кровяной солью K4[Fe(CN)6] — образуется синий осадок «берлинская лазурь».
  • С роданидами (KCNS, NaCNS) — раствор окрашивается в кроваво-красный цвет (образуется Fe(CNS)3):
    FeCl3 + 3NaCNS → Fe(CNS)3 + 3NaCl
Fe(OH)3 — бурый осадок (не растворяется в избытке щёлочи в растворе).
Лайфхак для ЕГЭ: Турнбулева синь и берлинская лазурь по строению оказались идентичными веществами — Fe4[Fe2(CN)6]3. Но качественные реакции различаются: Fe2+ + красная кровяная соль, а Fe3+ + жёлтая кровяная соль.

6. Химические свойства простого вещества

При обычных условиях железо малоактивно благодаря оксидной плёнке. Однако при нагревании, особенно в мелкораздробленном состоянии, оно активно реагирует почти со всеми неметаллами.

Взаимодействие с неметаллами

С галогенами Активные галогены (F2, Cl2, Br2) окисляют железо до +3:
2Fe + 3Cl2 t°→ 2FeCl3
А менее активный йод — только до +2:
Fe + I2 t°→ FeI2
С серой Образуется сульфид железа (II):
Fe + S t°→ FeS
С фосфором Образуется фосфид железа:
Fe + P t°→ FeP
С кислородом При горении на воздухе образуется железная окалина — смешанный оксид:
3Fe + 2O2 t°→ Fe3O4
При пропускании O2 через расплав возможно образование FeO:
2Fe + O2 t°→ 2FeO
Важно для ЕГЭ: Запомните закономерность — сильные окислители (Cl2, Br2, HNO3) окисляют Fe до +3, а слабые (I2, S, HCl, разб. H2SO4) — до +2.

Взаимодействие со сложными веществами

1. С водой:
При обычных условиях практически не реагирует. Раскалённое железо (700–900°С) реагирует с водяным паром:

3Fe + 4H2O 700-900°C→ Fe3O4 + 4H2↑

Во влажном воздухе медленно корродирует (ржавеет):

4Fe + 3O2 + 6H2O → 4Fe(OH)3

2. С кислотами-неокислителями (HCl, разб. H2SO4):
Растворяется с образованием солей Fe2+ и водорода:

Fe + 2HCl → FeCl2 + H2↑

3. С конц. H2SO4:
На холоде — пассивация! При нагревании реакция идёт, железо окисляется до +3:

2Fe + 6H2SO4(конц.) t°→ Fe2(SO4)3 + 3SO2↑ + 6H2O

4. С азотной кислотой:
Холодная концентрированная HNO3 также пассивирует железо. При нагревании:

Fe + 6HNO3(конц.) t°→ Fe(NO3)3 + 3NO2↑ + 3H2O

С разбавленной HNO3:

Fe + 4HNO3(разб.) → Fe(NO3)3 + NO↑ + 2H2O

С очень разбавленной HNO3 образуется нитрат аммония:

8Fe + 30HNO3(оч. разб.) → 8Fe(NO3)3 + 3NH4NO3 + 9H2O

5. С сильными окислителями в щелочной среде:
Железо может быть окислено до степени окисления +6 (ферраты):

Fe + 2KOH + 3KNO3 t°, сплавл.→ K2FeO4 + 3KNO2 + H2O

6. С солями менее активных металлов:
Железо вытесняет менее активные металлы из растворов их солей:

Fe + CuSO4 → FeSO4 + Cu↓

7. Взаимодействие Fe с солями Fe3+:
Металлическое железо восстанавливает Fe3+ до Fe2+:

2FeCl3 + Fe → 3FeCl2

Fe2(SO4)3 + Fe → 3FeSO4

7. Оксид железа (II) — FeO

Твёрдое чёрное вещество, нерастворимое в воде. Типичный основный оксид. Неустойчив — легко окисляется до соединений Fe(III).

Получение

  • Частичное восстановление Fe2O3 водородом:
    Fe2O3 + H2 t°→ 2FeO + H2O
  • Частичное восстановление угарным газом:
    Fe2O3 + CO t°→ 2FeO + CO2
  • Восстановление Fe2O3 железом:
    Fe2O3 + Fe t°→ 3FeO
  • Разложение гидроксида железа (II):
    Fe(OH)2 t°→ FeO + H2O

Химические свойства

1. С кислотными оксидами — образуются соли:

FeO + SO3 → FeSO4

2. С кислотами:

FeO + 2HCl → FeCl2 + H2O

3. С водой не реагирует.

4. Окисление сильными кислотами-окислителями (Fe2+ → Fe3+):

FeO + 4HNO3(конц.) → Fe(NO3)3 + NO2↑ + 2H2O

3FeO + 10HNO3(разб.) t°→ 3Fe(NO3)3 + NO↑ + 5H2O

5. Восстановление (FeO — слабый окислитель):

FeO + CO t°→ Fe + CO2

8. Оксид железа (III) — Fe2O3

Твёрдое красно-коричневое вещество, нерастворимое в воде. Амфотерный оксид (с преобладанием основных свойств).

Получение

  • Окисление FeO кислородом:
    4FeO + O2 t°→ 2Fe2O3
  • Разложение гидроксида железа (III):
    2Fe(OH)3 t°→ Fe2O3 + 3H2O

Химические свойства

1. С кислотами:

Fe2O3 + 6HNO3 → 2Fe(NO3)3 + 3H2O

2. С щелочами (в расплаве — проявляет кислотные свойства, образуя ферриты):

Fe2O3 + 2NaOH t°, сплавл.→ 2NaFeO2 + H2O

3. С водой не реагирует.

4. Окисление до ферратов сильными окислителями в щелочной среде:

Fe2O3 + 3KNO3 + 4KOH t°→ 2K2FeO4 + 3KNO2 + 2H2O

5. Восстановление (окислительные свойства):

Fe2O3 + 3CO t°→ 2Fe + 3CO2

Fe2O3 + 3H2 t°→ 2Fe + 3H2O

Алюмотермия:

Fe2O3 + 2Al t°→ 2Fe + Al2O3

Восстановление железом (только до FeO):

Fe2O3 + Fe t°→ 3FeO

6. Вытеснение летучих оксидов из солей при сплавлении:

Fe2O3 + Na2CO3 t°→ 2NaFeO2 + CO2↑

9. Оксид железа (II, III) — Fe3O4 (железная окалина)

Твёрдое чёрное вещество, нерастворимое в воде. Представляет собой смешанный оксид (FeO·Fe2O3), сочетающий свойства обоих оксидов. В природе — минерал магнетит.

Получение

  • Горение железа на воздухе:
    3Fe + 2O2 t°→ Fe3O4
  • Реакция раскалённого железа с водяным паром:
    3Fe + 4H2O(пар) t°→ Fe3O4 + 4H2↑

Химические свойства

1. С кислотами — образуются две соли (Fe2+ и Fe3+):

Fe3O4 + 8HCl → FeCl2 + 2FeCl3 + 4H2O

2. С кислотами-окислителями (всё железо окисляется до +3):

Fe3O4 + 10HNO3(конц.) → 3Fe(NO3)3 + NO2↑ + 5H2O

2Fe3O4 + 10H2SO4(конц.) → 3Fe2(SO4)3 + SO2↑ + 10H2O

Окисление кислородом:

4Fe3O4 + O2 t°→ 6Fe2O3

3. Восстановление:

Fe3O4 + 4CO t°→ 3Fe + 4CO2

Fe3O4 + 4H2 t°→ 3Fe + 4H2O

Алюмотермия:

3Fe3O4 + 8Al t°→ 9Fe + 4Al2O3

С йодоводородом (восстановитель HI сначала восстанавливает Fe3+ до Fe2+):

Fe3O4 + 8HI → 3FeI2 + I2 + 4H2O

10. Гидроксид железа (II) — Fe(OH)2

Белый (серо-зелёный на практике) студенистый осадок. Основный гидроксид. Слабое основание, нерастворимое в воде. На воздухе быстро буреет, окисляясь до Fe(OH)3.

Получение

  • Действие щёлочи на соли Fe(II):
    FeCl2 + 2KOH → Fe(OH)2↓ + 2KCl
  • Действие раствора аммиака на соли Fe(II):
    FeCl2 + 2NH3 + 2H2O → Fe(OH)2↓ + 2NH4Cl

Химические свойства

1. С кислотами (основные свойства):

Fe(OH)2 + 2HCl → FeCl2 + 2H2O

Fe(OH)2 + H2SO4 → FeSO4 + 2H2O

2. С кислотными оксидами сильных кислот:

Fe(OH)2 + SO3 → FeSO4 + 2H2O

3. Восстановительные свойства — легко окисляется до Fe(III):

4Fe(OH)2 + O2 + 2H2O → 4Fe(OH)3↓

2Fe(OH)2 + H2O2 → 2Fe(OH)3

При растворении в кислотах-окислителях образуются соли Fe(III):

2Fe(OH)2 + 4H2SO4(конц.) → Fe2(SO4)3 + SO2↑ + 6H2O

4. Разложение при нагревании:

Fe(OH)2 t°→ FeO + H2O

11. Гидроксид железа (III) — Fe(OH)3

Бурый аморфный осадок, нерастворимый в воде. Амфотерный гидроксид с преобладанием основных свойств. Кислотные свойства проявляет только при сплавлении с щелочами.

Получение

  • Действие щёлочи на соли Fe(III):
    FeCl3 + 3KOH → Fe(OH)3↓ + 3KCl
  • Действие раствора аммиака:
    FeCl3 + 3NH3 + 3H2O → Fe(OH)3↓ + 3NH4Cl
  • Окисление Fe(OH)2:
    4Fe(OH)2 + O2 + 2H2O → 4Fe(OH)3↓
  • Необратимый гидролиз при взаимодействии солей Fe(III) с карбонатами:
    2FeCl3 + 3Na2CO3 + 3H2O → 2Fe(OH)3↓ + 3CO2↑ + 6NaCl
Важно для ЕГЭ: Карбонат и сульфит железа (III) не существуют в водном растворе — они полностью и необратимо гидролизуются с образованием Fe(OH)3↓ и соответствующего газа (CO2 или SO2). Но с сульфитами чаще записывают ОВР — Fe3+ окисляет SO32−.

Химические свойства

1. С кислотами (основные свойства):

Fe(OH)3 + 3HCl → FeCl3 + 3H2O

Fe(OH)3 + 3HNO3 → Fe(NO3)3 + 3H2O

2Fe(OH)3 + 3H2SO4 → Fe2(SO4)3 + 6H2O

2. С кислотными оксидами сильных кислот:

2Fe(OH)3 + 3SO3 → Fe2(SO4)3 + 3H2O

3. С щелочами (кислотные свойства — только при сплавлении!):

Fe(OH)3 + KOH t°, сплавл.→ KFeO2 + 2H2O

4. Разложение при нагревании:

2Fe(OH)3 t°→ Fe2O3 + 3H2O

12. Соли железа

Термическое разложение нитратов

Нитраты железа разлагаются по общей схеме нитратов металлов средней активности — до оксида металла, NO2 и O2:

4Fe(NO3)2 t°→ 2Fe2O3 + 8NO2↑ + O2↑

4Fe(NO3)3 t°→ 2Fe2O3 + 12NO2↑ + 3O2↑

Гидролиз солей железа

Растворимые соли железа (III), образованные сильными кислотами, гидролизуются ступенчато по катиону (среда кислая). Карбонаты и сульфиты Fe(III) гидролизуются необратимо:

2FeCl3 + 3Na2CO3 + 3H2O → 2Fe(OH)3↓ + 6NaCl + 3CO2↑

Fe2(SO4)3 + 3K2CO3 + 3H2O → 2Fe(OH)3↓ + 3CO2↑ + 3K2SO4

Взаимодействие солей Fe(III) с сульфитами — ОВР (Fe3+ окисляет SO32−):

2FeCl3 + Na2SO3 + H2O → 2FeCl2 + Na2SO4 + 2HCl

Окислительные свойства солей Fe(III)

Соединения Fe3+ проявляют довольно сильные окислительные свойства — сами восстанавливаются до Fe2+.

С сульфидами:

2FeCl3 + 3Na2S → 2FeS + S + 6NaCl

2FeCl3 + H2S → 2FeCl2 + S↓ + 2HCl

С йодидами:

2FeCl3 + 2KI → 2FeCl2 + I2 + 2KCl

С металлами:
Соли Fe(III) могут реагировать даже с металлами, стоящими после железа в ряду активности (вплоть до меди включительно), восстанавливаясь до Fe2+:

2FeCl3 + Cu → 2FeCl2 + CuCl2

Более активные металлы (левее Fe в ряду активности) восстанавливают железо до простого вещества:

2Fe(NO3)3 + 3Zn → 2Fe + 3Zn(NO3)2

Лайфхак для ЕГЭ: Fe3+ — окислитель «средней силы». Он окисляет Cu, I−, S2−, SO32−. Но не окисляет Br− и Cl−! Запомните: FeCl3 растворяет медные платы — именно на этом основано травление печатных плат.
